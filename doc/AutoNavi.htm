关键字查询

l  接口功能介绍

基础数据关键字搜索。

l  接口参数说明

参数名称

参数格式

参数含义

备注

sid

sid=1000

服务编码

必填参数

rid

 

请求ID

可选参数

encode

encode=GBK/UTF-8

请求串编码，不区分大小写，默认值为GBK

可选参数

key

 

用户Key

必填参数

city

 

支持传入城市名称、城市区号、adcode（支持行政区划，省、市、区县）。默认查询全国数据。

可选参数

sort

sort=0/1/2/3/4

排序规则

0表示默认混合排序方式

1表示周边查询只按距离排序

2表示通用编辑器POI按名称字符串排序

3表示通用编辑器POI按字段长度排序

4表示通用编辑器POI按饱和度排序

可选参数

type

 

数据分类类型，如酒店、医院、学校、加油站等

可选参数

batch

 

当前显示页，默认为第一页

可选参数

number

 

每页记录数，默认为每页显示10条记录

可选参数

srctype

 

数据源

该参数定义了在哪些数据中执行查询，同时指定在查询多个来源的数据时不同来源的结果间的排序规则。

不区分大小写。

基础库：POI

企业地标：DIBIAO

公交数据：BUS

可对数据源进行混合查询，如，需要同时查询基础库与公交数据，并要求前1000条记录来自公交数据，后续记录来自基础库，可如下设置：

srctype=BUS:1000%2bPOI

默认值为POI

可选参数

keyword

 

关键字

必填参数

ext

 

扩展字段

作为过滤条件条件使用，格式如下："ext=custom(name:大恒;address:苏州街)-custom_and(false)"

其中， custom是定制的过滤条件，name和address是返回结果中的属性值，custom_and是custom的判断条件。当custom_and取值为true时，表示返回结果中既有“苏州街”又有“大恒”，当取值为false时，表示返回结果中有“苏州街”“大恒”中的一个即可。

可选参数

resType

resType=xml/json

返回结果的格式，支持XML与JSON，不区分大小写，默认为XML格式

可选参数

l  URL示例

http://<host:port>/gss/simple?sid=1000&resType=xml&batch=1&number=2&rid=123&keyword=%BE%C6%B5%EA&city=%B1%B1%BE%A9&key=<用户Key>

公交导航

l  接口功能介绍

通过起点、终点查询到达方式。

l  接口参数说明

公交导航接口参数说明

参数名称

参数格式

参数含义

备注

sid

sid=8001

服务编码

必填参数

city

 

城市区号，例如：北京为010

必填参数

key

 

用户Key

必填参数

xys

 

途经点坐标集合

必填参数

encode

encode=GBK/UTF-8

请求串编码，不区分大小写，默认值为GBK

可选参数

resType

resType=xml/json

返回结果的格式，支持XML与JSON，不区分大小写，默认为XML格式

可选参数

type

type=0/1/2/3/4/5/6/7

行驶类型

0表示最快捷模式，尽可能乘坐轨道交通和快速公交线路；

2表示最少换乘模式，尽可能减少换乘次数；

3表示最少步行模式，尽可能减少步行距离；

4表示最舒适模式，；乘坐有空调的车线；

5表示纯地铁模式，只选择地铁换乘；

 

 

l  URL示例

http://<host:port>/bus/simple?sid=8001&city=010&xys=116.4604213,39.9204703;116.2883602,39.8236433&resType=xml&rid=123&key=<用户Key>

路径导航服务

l  接口功能介绍

驾车路径导航。

l  接口参数说明

参数名称

参数格式

参数含义

备注

sid

sid=8000

服务编码

必填参数

rid

 

请求ID

可选参数

encode

encode=GBK/UTF-8

请求串编码，不区分大小写，默认值为GBK

可选参数

key

 

用户Key

必填参数

routeType

 

路径计算规则，默认为routeType=0

routeType=0速度优先（快速路优先）

routeType=1费用优先（不走收费路段的最快道路）

routeType=2距离优先（路径最短）

routeType=5不走快速路（不走快速路，不包含高速路）

routeType=6国道优先

routeType=7省道优先

routeType=9多策略（同时使用速度优先、费用优先、距离优先三个策略计算路径）

可选参数

xys

 

坐标对集合，支持13个途经点，多个途经点使用";"隔开，坐标点内部使用","隔开，格式如下：

x1,y1;x2,y2;…

必填参数

avoidRegion

 

区域避让，支持32个避让区域，每个区域最多可有16个顶点

避让区域之间使用":"区分，避让区域由坐标对组成，坐标对之间使用";"分隔，如果是四边形则有四个坐标点，如果是五边形则有五个坐标点

可选参数

avoidName

 

避让名称

如一条道路的名称

可选参数

per

 

抽稀参数，默认值为50

即返回途经点的坐标对个数

可选参数

resType

resType=xml/json

返回结果的格式，支持XML与JSON，不区分大小写，默认为XML格式

可选参数

 

l  URL示例

http://<host:port>/route/simple?sid=8000&resType=xml&xys=116.3160,39.96497;117.180445,39.2634875;117.170445,39.2534875;117.160445,39.2434875&per=150&routeType=1&rid=123&key=<用户Key>
